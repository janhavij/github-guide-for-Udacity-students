Creating a GitHub Account

In this guide, we will create a GitHub account. You might see some words/phrases that don’t yet make sense. Don’t worry! You will learn those later. Let us make an account first.
Click here to open the GitHub website. The webpage looks like this -
On the right side of the screen, you will see “Sign up for GitHub” button. Pick a username, email and password and click on “Sign up for GitHub”.
You will be taken to a “Welcome to GitHub page”. The page will show you different options for “paid subscriptions”. You do not need to choose any of those.
Click on “Finish sign up” at the right bottom of the screen. You will see the following on your screen.
Congratulations! You now have a GitHub account. Next, we will show you how to upload your project on GitHub.
