Setting up GitHub on Your Mac

An Easy Guide to Getting Your Project on GitHub

Click here to visit the GitHub for Mac website.
Click the "Download GitHub for Mac" button.
This will download two files in your “Downloads” folder (or the folder where all your downloaded files get stored.)
Navigate to the folder where the file was downloaded. And open the file named “GitHub”.
Your computer will show you the following prompt to ensure that you want to open this application.
Click on “Open”. Your computer will ask you if you want to move this application to the Applications folder. Click on “Move to Applications Folder”.
You will next see the “Thanks for trying GitHub for Mac!” on your screen. Click “Continue”.
This will take you to the following Login screen. Enter your GitHub username and password and click “Sign In”.
Next we will Configure Git. Don’t worry if you don’t yet understand what these terms mean. We will learn these later. Enter your Name and Email Address as shown below and click “Continue”.
Next screen will ask you “Which repositories you would like to add to GitHub for Mac”. Click Continue and you will see the following screen.
Next, In your finder, navigate to the folder that has your project files and “drag and drop” the folder on GitHub for Mac. You should see the following prompt on the screen.
Click “Create and Add”. Your screen will look similar to the following.
Click on “Publish” - which is at the top right corner of the GitHub for Mac application. Following prompt to login into GitHub will appear on the screen. Click OK and enter your login and password as shown below.
Next you will see the following screen. Click on the red button on the top left and close it.
You will be prompted to enter the “Description” for your project. Enter the appropriate description and click on “Push Repository”.
Congratulations! You are ready to submit your project now. We look forward to your success! If you have any problems, please don't hesitate to email us at review-support@udacity.com.
